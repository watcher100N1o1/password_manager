## Описание программы:

Password Manager - консольное приложение, для хранения, генерации и управления паролями. Пароли и все идущие вместе с ними данные сохраняются в файле data.json
Создается как проект для итоговой аттестаци по курсу программирования.


## Требования для запуска
Что нужно установить на компьютер:
- Python 3.10 или выше
- Библиотеки: `pip install json` (если ещё не установлены)

## Как запустить

bash:
- git clone https://github.com/watcher100N1o1/password_manager.git
- cd password_manager
- python main.py

или
- python3 main.py

# Как пользоваться:
При запуске программы будет выведено меню

 Выберите действие: 
 
1 - создать новую запись

2 - удалить запись

3 - просмотр записей

4 - записать в файл

5 - прочитать файл

6 - выход 

Выберите одну из комманд, введя её номер в поле ввода, обозначенное ">"
Программа будет работать по одному из этих сценариев:

## 1 - создать новую запись
  Будут последовательно предоставленны 3 поля ввода:
  
> Введите имя аккаунта: (что угодно)
 
> Введите название сервиса/ссылку на сервис: (что угодно)
 
> Введите длинну пароля: (только число)

> Добавить заметку(Д/Н): (l/д/любой другой символ(ы))

  Если выбрали "д" или "l", будет выведено поле ввода:

> Введите текст заметки: (текст)

  Если ввели любой другой, поля ввода не выведется

  Введите соответствующую информацию и нажимайте enter чтобы перейти к следующему полю.

  После всех вводов запись будет создана.



## 2 - удалить запись
 Будет выведено поле ввода:
 
> Введите юзернейм: (имя аккаунта, запись о котором вы хотите удалить)

 Введите имя (юзернейм) и нажмите enter

 Если аккаунт с таким именем есть:

 Будет выведено поле ввода:
> Вы уверены, что хотите удалить запись(Д/Н):
> (l/д/любой другой символ)

 Если введёте "д" или "l" - выведется сообщение об удалении записи:
> Запись удалена!
 
 сама запись будет удалена
 
 Если лююой другой символ - действие будет отменено, а также будет выведено сообщение:
> Отмена удаления...
 

## 3 - просмотр записей
 Будет выведен список записей в подобном формате:
 
 
> Юзернейм: юзер
> 
> Сервис: серв
> 
> Пароль: пароль
> 
> Дата создания: 11.11.1111 00:00
> 
> (==========)
 
или 

> Юзернейм: юзер
> 
> Сервис: серв
> 
> Пароль: пароль
> 
> Заметка: 
> текст_заметки
>
> Дата создания: 11.11.1111 00:00
> 
> (==========)


 Если записей нет, будет выведено сообщение:
> Список записей пуст.

## 4 - записать в файл
 Будут выведены сообщения:
> Данные получены.
> Файл обновлен.

 А файл будет дополнен записями, которые были в списке, но не в файле. 


## 5 - прочитать файл
 Если файл существует:

 Будет выведено сообщение:
 
> Данные получены.

 Если файла несуществует:
 
 Будут выведены сообщения:
 
> Ошибка: файл несуществует!
>
> Создание файла...

 После чего будет создан  пустой файл.

## 6 - выход
 Будут выведены сообщения:

> Завершение программы...
>
> Всего доброго!

## В случае ввода непредусмотренной комманды:
 Будут выведены сообщения:

>Ошибка: неверная команда!
>
>Попробуйте ввести одну из предложенных комманд.

### После успешного или нет выполнения комманды, кроме последней, вас вернёт в меню выбора комманды.

## В случае "грубого" завершения программы:
 Будет выведено сообщение:
> Завершение программы...
