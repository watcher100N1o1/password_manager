## Описание программы

Password Manager - консольное приложение, для хранения, генерации и управления паролями.
Создается как проект для итоговой аттестаци по курсу программирования.


## Требования для запуска
Что нужно установить на компьютер:
- Python 3.10 или выше
- Библиотеки: `pip install matplotlib requests`

## Как запустить

bash:
- git clone https://github.com/ watcher100N1o1/final_exam_project.git
- cd final_exam_project
